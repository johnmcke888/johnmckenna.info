Valyou Financial Dashboard
A single-input financial awareness tool that translates dollar amounts into multiple life contexts simultaneously. Enter any amount and instantly see what it represents in hours worked, investment portfolio impact, goal progress, and debt payoff potential. Built to make money feel less abstract through brutalist design and real-time context translation across four pillars: Earnings, Investments, Goals, and Debts.

Promotional Value Modeler
A prescriptive modeling engine that analyzes casino promotional mechanics and outputs exact, quantified strategies for completing offers. Takes promotion parameters as input and provides a complete plan of optimal deposit amounts, precise wager sizes, specific bet selections, and guaranteed outcomes—no guesswork, no luck-based recommendations. Evolved from basic Excel modeling to a sophisticated React application handling multi-promo scenarios and complex optimization logic.
Tags: Prescriptive Modeling Expected Value Analysis Constrained Optimization React
Balloon Betting Modeler
A risk profile transformation tool that leverages casino bonus value to completely reshape gambling risk. Uses sequential wagering strategy to progressively increase bet sizes while maintaining mathematical break-even positioning. The app dynamically recalculates optimal balloon bet sizes at each decision point along a probabilistic decision tree, providing real-time metrics on profit ranges, wagering progress, and risk probability to enable informed decision-making in live play scenarios.
Tags: Risk Profile Transformation Sequential Wagering Strategy Decision Tree Modeling Break-Even Calculation
Fogo Family Fleet Tracker
A comprehensive rewards lifecycle management system for coordinating Fogo de Chão dining across multiple people. Tracks reward status (Locked/Pending/Available) for each family member, logs every visit with detailed spend and discount analysis, manages gift card balances with automated email parsing, and calculates true meal costs factoring in stacked rewards and promotional discounts. Features a high-density spreadsheet view showing visit history with real-time "% Saved" metrics that gamify the optimization process.
Tags: Rewards Lifecycle Tracking Gift Card Management Multi-Person Coordination Data Visualization
Live Dealer tStart Analysis
A statistical research project investigating whether online live dealer baccarat manipulates the betting window duration (tStart) based on player betting patterns. Built custom data collection tools to capture timing metadata invisible to standard players, conducted rigorous hypothesis testing using Mann-Whitney U tests and effect size analysis, and discovered that initial per-hand manipulation theory was misspecified. Currently developing regime-based analysis using change point detection to test whether tStart operates in sustained fast/slow periods correlated with house-favorable outcomes.
Tags: Statistical Analysis Hypothesis Testing Data Collection Research Methodology Python
