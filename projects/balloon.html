Balloon Betting Modeler

An analytical methodology that reshapes the risk profile of gambling, setting a floor for the downside while preserving exponential upside.

The Gambler's Dilemma

In traditional gambling, the potential for exponential gains is offset by the potential for exponential losses. Due to the house edge, the probability of losses is always greater, which is why systems like the Martingale are so infamous. One bad streak can wipe out all previous gains.

This inspired me to develop a new strategy leveraging casino bonuses. Could I use the bonus value to completely reshape this risk profile? What if there was a strategy that eliminated the risk of downside (loss of funds) while maintaining the upside (potential for exponential returns from a winning streak)?

The "Balloon" Methodology

The result was a methodology I named "Balloon Betting." As a balloon is filled with air, there is a risk that it will pop. The more air filled into a balloon, the more the chance that it will pop converges to 100%. The goal is to inflate it enough to look nice, but stop before it pops.

This tool models that concept. It calculates the maximum, "safe" bet size for each step—an amount you can wager where a loss (a "pop") still lets you exit with $0 profit, breaking even, but importantly not losing money. A win inflates the balloon, and the tool recalculates a new, larger "safe" bet, all while tracking progress against the bonus wagering requirements.

Project Details

Like the main promo analyzer, this tool evolved from a basic spreadsheet into a full-featured React app. Because the payout of Player and Banker bets is different, the app must dynamically recalculate the optimal "balloon" bet size at each point on the decision tree, all while tracking progress against wagering requirements.

Stack: React, JavaScript (Hooks), Babel, Tailwind CSS

Core Concepts: Risk Profile Modeling, Sequential Analysis, Break-Even Calculation, Decision Tree Modeling

Status: Personal R&D (Unpublished)

Concept Development

The first tool was a spreadsheet that proved the concept. It was possible to leverage casino bonus value to completely reshape the risk profile of gambling. However, it was basic and assumed a series of player bets only.

Basic Implementation (v1.0)

The first working implementation in React dynamically calculated the appropriate balloon bet size along the way as the user selected Player or Banker, solving the spreadsheet's limitation.

Single Casino Mode (v1.5)

Later, a "Single Casino" mode was introduced, proving the logic could stand alone. This included a "High Risk" mode where the user was liable to lose their deposit, but only on the first round.

Dynamic Tracking (v2.0)

This version focused on providing metrics for informed decisions, including a wagering progress bar and a probability meter. Seeing the probability get cut in half with each step drove home the risk of continuing.

Profit Evolution Table (v2.10)

Implemented a profit evolution table showing profit ranges that narrow with each bet. All-Player wins provided the ceiling, all-Banker wins provided the floor, and the range converged on an actual figure as the user played.

Custom Bet Feature (v2.15)

I implemented a "Custom Bet" feature that allowed users to account for side bets or to place a smaller bet than the "optimal" size if they wanted to keep a profit buffer in case of a loss.

Current Version (v2.19)

The current version is a production-ready tool with full session management (reset, undo) and a polished, responsive UI for quick, on-the-fly decision making.

Future Development

This methodology is highly portable. I am in the process of adapting it for other games that offer ~50/50 odds, like Blackjack, Craps, or Roulette. The core logic is game-agnostic—it works anywhere there are near-even odds and sequential decisions.

I am also developing a "sister" strategy called Moon Betting, which explores a similar risk-reshaping profile but does not require a casino bonus to work, instead capping the user's capital risk at a fixed amount.

Project Philosophy & Motivation

This tool was built to provide authentic gamblers with structure and guidance. By giving clear bet sizes and an exit point, it's an exercise in analytical game design—a way to provide a smarter, safer framework for the very real thrill of chasing a win streak. For me, solving the puzzle is the real reward.
